# Clima 2.0 — Previsão do Tempo Interativa

> App web de **previsão do tempo** com busca por cidade e previsão para os próximos dias, usando a WeatherAPI.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

Aplicação web em **JavaScript puro (vanilla)** que consome a API da [WeatherAPI](https://www.weatherapi.com/) para mostrar o clima atual e a **previsão de 7 dias** de qualquer cidade, com temperatura, umidade, vento e ícones do tempo. Interface responsiva e em português.

## ✨ Funcionalidades

- 🔍 **Busca por cidade**
- 🌡️ Clima atual: **temperatura, umidade e vento**
- 📅 **Previsão de 7 dias**, com mínimas/máximas e ícones
- 🌤️ Ícones do tempo conforme a condição
- 📱 Layout responsivo (pensado também para mobile)

## 🚀 Como usar

1. Crie uma conta gratuita em **[weatherapi.com](https://www.weatherapi.com/)** e copie sua **API key**.
2. No arquivo **`script.js`**, coloque sua chave na constante:
   ```js
   const apiKey = "SUA_CHAVE_AQUI";
   ```
3. Abra o **`index.html`** no navegador (ou use o *Live Server* do VS Code).
4. Digite o nome de uma cidade e veja a previsão.

> ⚠️ A chave da API é pessoal — não a compartilhe publicamente.

## 🛠️ Tecnologias

- **HTML5** · **CSS3** · **JavaScript (ES6, async/await, fetch)**
- **WeatherAPI** (API REST de previsão do tempo)
